# 薬局情報のマップ表示

[github page](https://tobizaru.github.io/pharmacy_map)にて公開中

厚生労働省の各厚生局HPから取得した薬局の施設基準の届出状況より
調剤基本料を計算しマップ上に表示します。

## 使用言語

Vue 3

## インストール方法

1. NodeJSインストール
1. $ git clone https://github.com/tobizaru/pharmacy_map
1. $ cd frontend
1. $ npm install
2. $ npm run dev
3. http://localhost:3000にアクセス
